# ⚡ Meu Fluxo | Gerenciador de Tarefas 

Este projeto é um gerenciador de tarefas desenvolvido para ser prático, rápido e visualmente moderno. Foquei em criar uma interface de alto contraste que une estética e funcionalidade, garantindo que a organização do dia a dia seja simples e intuitiva.

### 🚀 Funcionalidades Principais

* **Interface Adaptável:** Suporte total a Modo Escuro (Dark Mode). O app identifica e salva sua preferência via LocalStorage, mantendo o visual escolhido mesmo após fechar o navegador.
* **Monitoramento de Progresso:** Dashboard integrado que exibe o total de tarefas, pendências e itens concluídos em tempo real.
* **Filtros Dinâmicos:** Sistema de busca por texto e filtros por status (Todas, Pendentes e Concluídas) para facilitar a navegação.
* **Persistência de Dados:** Integração com `LocalStorage`, garantindo que sua lista esteja sempre disponível em acessos futuros.
* **Design Polido:** Acabamento visual moderno com animações de transição e elementos de destaque que tornam a experiência mais agradável.

### 🛠️ Tecnologias Utilizadas

O projeto foi construído do zero, focando em conceitos sólidos de desenvolvimento web:

* **HTML5** para uma estrutura semântica e organizada.
* **CSS3** avançado, utilizando variáveis (Custom Properties) e animações para um layout fluido e responsivo.
* **JavaScript (Vanilla)** para toda a lógica de manipulação de dados, gerenciamento de estados e interatividade.

### 📂 Como usar

1.  Clone ou baixe os arquivos deste repositório.
2.  Abra o arquivo `index.html` em seu navegador de preferência.
3.  Pronto! Suas tarefas e preferências de tema já estarão sendo salvas automaticamente no seu navegador.
